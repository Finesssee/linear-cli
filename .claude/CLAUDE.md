# linear-cli

CLI tool for Linear.app project management.

## Stack
- Rust
- Source in `src/`

## Commands
- `cargo build --release` - build
- `cargo test` - run tests
- `cargo clippy` - lint (CI fails on warnings)
- `cargo fmt` - format

## Usage
Binary at `target/release/linear-cli.exe`

## Key Features
- Projects, Issues, Labels, Teams, Users, Cycles
- Git branch operations
- Sync with D:\code folders
